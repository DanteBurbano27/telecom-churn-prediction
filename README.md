# Telecom Churn Prediction

This project applies machine learning to predict customer churn in a telecom business context.

The objective is not only to classify churn risk, but also to translate model outputs into actionable retention strategies.

## Business Problem

Telecom companies face revenue loss when customers cancel or stop using their services.

This project identifies high-risk customers and prioritizes intervention based on churn probability and customer value.

## Project Objectives

- Explore customer behavior patterns
- Build a churn prediction model
- Evaluate model performance using classification metrics
- Segment customers by churn risk
- Estimate business impact through targeted retention actions

## Workflow

1. Data understanding
2. Exploratory Data Analysis
3. Data cleaning and preprocessing
4. Feature engineering
5. Model training
6. Model evaluation
7. Risk segmentation
8. Business recommendation

## Machine Learning Models

The project may include models such as:

- Logistic Regression
- Random Forest
- Gradient Boosting
- XGBoost or similar tree-based models

## Evaluation Metrics

- ROC-AUC
- Precision
- Recall
- F1-score
- Confusion Matrix
- Business impact by risk decile

## Business Value

The model supports a targeted retention strategy by identifying customers with high churn probability and prioritizing them according to potential revenue impact.
